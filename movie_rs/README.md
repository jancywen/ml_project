### 基于内存的协同过滤推荐

1. ICF

构建UI矩阵计算物品之间的相似度,根据物品的相似度和用户的历史行为给用户生成推荐列表.

2. UCF

构建UI矩阵计算用户之间的相似度,根据用户的相似度和用户的历史行为给用户生成推荐列表.
